Prototypes

We continuously test designs by building prototypes, both using this styleguide but also using other means. This page is supposed to help you navigate all of these efforts by providing pointers to relevant resources. Some of these pages are linked together, so there might be more than one way of getting to any given prototype. Sometimes the pages look funky or broken, they are just prototypes after all, built by designers hacking away.

Built using styleguide

  • Homepage - An example of a customer homepage
  • Site Appearance - Where admins would customize their site
  • Catalog - An example of a generic catalog
    • Catalog Facet - AKA Category page, Category Hub or CHUB
    • Catalog Rich View - A results page featuring only visualizations and stories
  • Dataset Landing Page - The page that is a dataset
    • Dataset Landing Page - Edit mode - How a publisher would edit their Landing Page
