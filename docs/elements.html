Typography

Type is set in Open Sans 15px/1.618 using a modular type scale. The scale is composed using three scale ratios to provide more incremental typesetting steps.

The full scale can be viewed at http://www.modularscale.com/?15&px&1.618,1.25,1.33

Specimens

Open Sans: AaBbCc 1234 Open Sans Light Open Sans: AaBbCc 1234 Open Sans Light Italic

Open Sans: AaBbCc 1234 Open Sans Open Sans: AaBbCc 1234 Open Sans Italic

Open Sans: AaBbCc 1234 Open Sans Semibold Open Sans: AaBbCc 1234 Open Sans Semibold Italic

Open Sans: AaBbCc 1234 Open Sans Bold Open Sans: AaBbCc 1234 Open Sans Bold Italic

      <li>I'm also nested.</li>
    </ul>
  </li>
  <li>We're still going.</li>
</ul>

Ordered List

  1. This is one more
  2. simply, absolutely fantastic
  3. This is a nested ordered list!
    1. I'm nested!
    2. I'm also nested.
    3. More nests!
      1. I'm über nested.
      2. What that ^ <li> said.
  4. We're still going.

<ol>
  <li>This is one more</li>
  <li>simply, absolutely fantastic</li>
  <li>This is a nested ordered list!
    <ol>

      <li>More nests!
        <ol>

        </ol>
      </li>
    </ol>
  </li>
  <li>We're still going.</li>
</ol>

Alerts

Alerts draw the user's attention to a specific condition, either prior to or in response to a user action.

The type of alert embodies the sentiment of the condition; the content reinforces good/expected behaviors or provides actionable information for bad/unexpected behaviors.

Success
